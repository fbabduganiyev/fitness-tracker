DAILY TRACKER — deploy to Vercel (free)

You do NOT need to install anything on your computer. Everything happens in the browser.

── STEP 1: Make a GitHub account (skip if you have one) ──
Go to github.com and sign up (free).

── STEP 2: Put this project on GitHub ──
1. On github.com, click the "+" (top right) → "New repository".
2. Name it "fitness-tracker", keep it Public, click "Create repository".
3. On the new repo page, click "uploading an existing file".
4. Unzip this folder on your computer, then drag ALL the files
   (package.json, index.html, vite.config.js, and the src folder) into the browser.
5. Click "Commit changes".

── STEP 3: Deploy on Vercel ──
1. Go to vercel.com → "Sign Up" → choose "Continue with GitHub".
2. Click "Add New..." → "Project".
3. Find "fitness-tracker" in the list → click "Import".
4. Leave every setting as-is (Vercel auto-detects Vite) → click "Deploy".
5. Wait ~1 minute. You'll get a live link like fitness-tracker-xxx.vercel.app

── STEP 4: Put it on your iPhone home screen ──
1. Open that link in Safari on your iPhone.
2. Tap the Share button (square with an up-arrow at the bottom).
3. Scroll down → "Add to Home Screen" → "Add".
4. Done — tap the new icon and it opens full-screen like a real app.

── NOTES ──
- Your data (streak, weight, meals) saves on the phone/browser you use.
  Use the same one each time or your history won't follow you.
- The app is tuned to your targets: 2700 kcal, 120g protein.
- To change anything later, edit the file on GitHub and Vercel
  auto-redeploys within a minute.
